Success Criterion 2.4.9: Link Purpose (Link Only) (Level AAA)

A mechanism is available to allow the purpose of each link to be identified from link text alone, except where the purpose of the link would be ambiguous to users in general.

(From Understanding WCAG 2.0: Understanding Success Criterion 2.4.9)

Discussion

Summary

This is a stricter version of Success Criterion 2.4.4. At Level AAA, the immediate surrounding context around a link - such as the remaining text in a sentence, cell or list item, or the previous header or list - cannot be used for claiming that link text describes the purpose of the link. These techniques remain as advisory rather than sufficient, and thus are still recommended to improve accessibility.
