Success Criterion 2.4.5: Multiple Ways (Level AA)

More than one way is available to locate a Web page within a set of Web pages except where the Web Page is the result of, or a step in, a process.

(From Understanding WCAG 2.0: Understanding Success Criterion 2.4.5)

Discussion

Summary

This Success Criterion aims to provide multiple methods of navigation - at least two - so that users that have issues with the default method of navigation have alternatives available.

Sufficient techniques include:

  • Links to related web pages (G125);
  • A Table of Contents (G64);
  • A site map (G63);
  • A search function (G161);
  • A full list of web pages (G126);
  • Linking to all other pages from the home page (G185);

Also, bookmarks in PDF documents are also considered sufficient for that technology. At least two of these need to be satisfied for this Success Criterion to be satisfied.

Some of these techniques are only useful for certain-sized sites. Linking to all pages in the web site, for instance, would be inappropriate on sites with a large number of pages - typically, if the size of the link list would be larger than the content itself.
