H24 Absent Area Alt Attribute Test

Explanation

According to Success Criterion 1.1.1 (Non-Text Content), all client-side image maps, using the map element, should have alt text (a short-text alternative) for each of its constituent area elements.

This example demonstrates an image map with two areas, both containing alt elements as specified. A notice is emitted to remind authors of the responsibility of ensuring the contents of the area element "serves the same purpose as the part of image map image referenced by the area element". In this example, these alternatives are probably not adequate.

Expected Not To Fail

  • [Standard].Principle1.Guideline1_1.1_1_1.H24: "Area element in an image map missing an alt attribute. Each area element must have a text alternative that describes the function of the image map area."

Expected Notices

  • [Standard].Principle1.Guideline1_1.1_1_1.H24.2: "Ensure that the area element's text alternative serves the same purpose as the part of image map image it references."

Relevant Success Criteria
