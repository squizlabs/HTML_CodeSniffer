H53 Absent Object Tag Alternative (Multiple Versions) Failure

Explanation

According to Success Criterion 1.1.1 (Non-text Content), all non-text content, including embedded objects, should contain a text alternative.

This example demonstrates the usage of both text and non-text alternatives - in this case, a static placeholder image if the QuickTime movie cannot be loaded. However, it should emit an error for having an empty text alternative, which should be in the body of the innermost element. The error should fire on the innermost object.
