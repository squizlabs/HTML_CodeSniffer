H67 NBSP Title on Ignored Image Failure

Explanation

An img element must follow the guidelines of Success Criterion 1.1.1 (Non-text Content) with regards usage of alt and title attributes.

This example uses empty alt text, but also provides a title attribute that is not considered empty. Non-breaking spaces are not counted as whitespace for the purpose of whether the title attribute is empty.

Expected Errors

  • [Standard].Principle1.Guideline1_1.1_1_1.H67.1: "Img element with empty alt text must have absent or empty title attribute."

Relevant Success Criteria

  • SC 1.1.1: Non-Text Content

Relevant Techniques

  • H67: Using null alt text and no title attribute on img elements for images that AT should ignore
