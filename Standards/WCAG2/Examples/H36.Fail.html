H36 Absent Alt Attribute On Image Submit Failure

Explanation

According to Success Criterion 1.1.1 (Non-text Content), all non-text content, including images, should contain a text alternative.

This example demonstrates an image submit button without an alt attribute. Without this, a user requiring assistance may not be able to submit the form properly, if their assistive technology does not allow them to submit the form another way. In this instance, it is similar to an image link with no alt text.

Expected Errors

  • [Standard].Principle1.Guideline1_1.1_1_1.H36: "Image submit button missing an alt attribute. Specify a text alternative that describes the button's function, using the alt attribute."

Relevant Success Criteria

  • SC 1.1.1: Non-Text Content
