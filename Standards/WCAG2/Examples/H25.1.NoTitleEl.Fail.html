H25.1 Missing Title Element Fail Example

Documents should be properly titled as per Success Criterion 2.4.2 (Page Titled). This test handles a situation where there is no title tag, which is necessary for a page to actually be titled.

Expected Errors

  • [Standard].Principle2.Guideline2_4.2_4_2.H25.1.NoTitleEl: "A title should be provided for the document, using a non-empty title element in the head section."

Relevant Success Criteria

  • SC 2.4.2: Page Titled

Relevant Techniques

  • H25: Providing a title using the title element
