H48.2 Lack of Ordered List Markup Warning

1. First Item
2. Second Item

1: First Item
2: Second Item

1 First Item
2 Second Item

1/ First Item
2/ Second Item

1 First Item
2 Second Item

Explanation

According to Success Criterion 1.3.1 (Info and Relationships), authors should use semantic markup wherever possible. This includes usage of ordered lists instead of manually numbered paragraphs.

This example should throw five warnings requesting that an ordered list be built out of the paragraph.

Expected Warnings

  • [Standard].Principle1.Guideline1_3.1_3_1.H48.2: "Content appears to have the visual appearance of a numbered list. It may be appropriate to mark this content up using an ol element."

Relevant Success Criteria

  • SC 1.3.1: Info and Relationships

Relevant Techniques

  • H48: Using ol, ul and dl for lists or groups of links
