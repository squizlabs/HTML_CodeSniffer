Explanation

According to Success Criterion 1.3.1 (Info and Relationships), providing a caption and/or summary can improve the usability of a table in different ways. A caption element titles the table and is seen by all users. A summary attribute (on the table element) provides additional information for users requiring assistance, who may be reading the table cell-by-cell and therefore cannot immediately grasp the concept of the table's organisation as a whole.

This example demonstrates incorrect usage of the summary element where there is also a caption - they should not duplicate each other. If they are the same, the summary then provides no further information to users requiring assistance, and is likely to still be inadequate to them.

Expected Errors

  • [Standard].Principle1.Guideline1_3.1_3_1.H39,H73.4: "If this table is a data table, and both a summary attribute and a caption element are present, the summary should not duplicate the caption."
