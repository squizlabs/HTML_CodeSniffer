H53 Absent Object Tag Alternative Failure

Explanation

According to Success Criterion 1.1.1 (Non-text Content), all non-text content, including embedded objects, should contain a text alternative.

This example should emit an error for having an empty alternative, which for object elements should be in the body of the element.

Expected Errors

  • [Standard].Principle1.Guideline1_1.1_1_1.H53: "Object elements must contain a text alternative after all other alternatives are exhausted."

Relevant Success Criteria

  • SC 1.1.1: Non-Text Content

Relevant Techniques

  • H53: Using the body of the object element
