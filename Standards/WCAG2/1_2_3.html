Success Criterion 1.2.3: Audio Description or Media Alternative (Prerecorded) (Level A)

An alternative for time-based media or audio description of the prerecorded video content is provided for synchronized media, except when the media is a media alternative for text and is clearly labeled as such.

(From Understanding WCAG 2.0: Understanding Success Criterion 1.2.3)

Note

Superseded at level AAA by both Success Criterion 1.2.5 and Success Criterion 1.2.8. Superseded at level AA by Success Criterion 1.2.5. Implementing these would satisfy this Success Criterion automatically.

Discussion

Definitions

For explanation of the definitions of real-time media used in the description of this Success Criterion, please see the discussion section of Success Criterion 1.2.1.

Superceding at higher conformance levels

The requirements of this Success Criterion are effectively a combination of Success Criterion 1.2.5, which requires an audio description of the video component, and Success Criterion 1.2.8, which requires an extended text alternative. Either can be chosen to satisfy this criterion at the A conformance level, while using both will help satisfy the AAA level.

There is a slight anomaly at the AA level, however. If targeting AA, it is recommended that you satisfy this Success Criterion using an audio description, because this will then satisfy Success Criterion 1.2.5, which applies at that level. The other criterion only applies at AAA level, and so only providing an extended text description, and not an audio description, will mean the web page would not conform to AA standard.
